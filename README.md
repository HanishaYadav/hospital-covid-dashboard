# Hospital COVID Dashboard

Interactive healthcare analytics dashboard built using Tableau and PostgreSQL.

## Project Overview

This project analyzes COVID hospital data across different states using interactive Tableau dashboards.

## Features

* KPI Overview
* Monthly Trend Analysis
* ICU Occupancy by State
* Capacity Analysis

## Tools Used

* Tableau
* PostgreSQL
* CSV Data Analysis
